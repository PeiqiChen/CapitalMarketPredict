# Library System

This is implentation of our Library System.

**Code structure**

```
|-- templates
|   |-- html
|
|-- bookSystem.ipynb
|
|-- book.csv
```

**How to run**

For prepare, run `bookSystem.ipynb` to grab data and insert into PostgreSQL.

In Browser, input `127.0.0.1:5001` to browse the books and authors' information.
